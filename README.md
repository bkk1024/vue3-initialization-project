# vue3初始化项目

#### 介绍

使用 vue3 + vite + pinia + js + vue-router + axios 初始化的项目构建

`pnpm install`

`pnpm dev`

#### 基础配置

1. `src` 目录配置别名 `@`
2. 配置 `defineOptions`
3. 配饰 `postcssPresetEnv()` postcss 预设环境、
4. 添加svg-icon组件

#### 开发配置

1. proxy 代理跨域
2. css 资源索引

#### 生产配置

1. base 路径
2. `node_modules` 分包
3. 打包去除 console 和 debugger
